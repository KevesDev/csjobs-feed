CS Jobs RSS Feed

An automated RSS feed aggregating remote computer science and programming job postings.

About This Project

This project fetches live job postings from a public jobs board, normalizes the data, and publishes a clean RSS 2.0 feed. The feed is regenerated automatically every 12 hours using GitHub Actions.

The goal is to demonstrate backend automation, data transformation, and static deployment using GitHub Pages.
